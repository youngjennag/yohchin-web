リセットCSSを使ってみよう

リセットCSSとは

リセットCSSとは、ブラウザにデフォルトで設定されているスタイルを打ち消したり調整したりして、ブラウザによる見え方の差をなくしてくれるCSSのことです。

リセットCSSの種類

  • リセットCSS系
  • ノーマライズCSS/サニタイズCSS系

よく使われるリセットCSS

  • destyle.css
  • ress.css
  • sanitize.css
  • Normalize.css

リセットCSSの使い方

  • CSSファイルをダウンロードして使う方法
  • CDN経由で使う方法
